# Evidence Policy

This file is a minimal repo-local placeholder for evidence expectations.

Current local rules:

- Evidence Packs should map each acceptance criterion to a concrete result
- artifacts should point to repo paths, test runs, or other check outputs
- unresolved risks should stay explicit until they are accepted or cleared

This stub is intentionally small. It records local expectations without making
hosted workflow behavior or product-specific tooling part of CCP v0.1.
