# Review Policy

This file is a minimal repo-local placeholder for review policy. It adds local
process notes without changing CCP core semantics.

Current local rules:

- substantial changes should start from an approved Change Request
- machine validation should pass before merge to `main`
- final closeout should include a linked Evidence Pack for the accepted change

Teams can replace this stub with stricter review rules later, but they should
keep the underlying CCP object lifecycle intact.
